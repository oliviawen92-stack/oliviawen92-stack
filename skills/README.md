# Skill Index

## Ready

### book-to-feishu-visual-notes

Repository: `oliviawen92-stack/book-to-feishu-visual-notes`

Turns books into Feishu/Lark visual reading notes:

- checks whether full book content is available,
- produces a public-information framework version when only public sources are available,
- creates 5-8 core viewpoints,
- creates one editable whiteboard per viewpoint,
- writes the result into a Feishu/Lark document.

## Ideas

- `feishu-briefing-board`: turn a topic or messy notes into a Feishu doc plus whiteboard explainer.
- `personal-wiki-curator`: ingest notes/files and maintain a personal knowledge index.
- `meeting-to-decision-log`: turn meeting notes into decisions, risks, and action items.
