# Olivia 的 Agent Skills

这里是我的个人 Agent Skill 空间，用来集中展示和维护我常用的可复用技能。

目标不是收集一堆炫技 demo，而是把真实工作和生活里反复出现的任务沉淀成稳定流程：读书、整理资料、生成飞书文档、做可编辑画板、输出结构化报告。

## Skills

| Skill | 用途 | 状态 |
| --- | --- | --- |
| [book-to-feishu-visual-notes](https://github.com/oliviawen92-stack/book-to-feishu-visual-notes) | 把来不及读完的书整理成飞书读书文档：核心观点、注释正文、每个观点一张可编辑白板。会明确区分“有完整书源”和“只有公开资料”的情况。 | 准备发布 |

## 设计原则

- **先说明信息边界。** 如果没有完整 PDF、EPUB、摘录或笔记，就不能假装读过全文，只能做公开信息框架版。
- **产出要可编辑。** 优先生成飞书文档、飞书画板、Markdown 等后续可继续编辑的材料。
- **一张图讲一个观点。** 不把一本书或一个复杂主题硬压成一张巨型图。
- **默认好用，但不装懂。** 先给出可用版本，同时说明如果补充什么材料，质量会更高。
